# uhcut

A modern web-based video and audio editing tool built with Angular and Capacitor.

## How to run on PC (Local Development)

To run the application locally on your PC, follow these steps:

1. **Install Dependencies:**
   Ensure you have Node.js and npm installed. Then, install the UI dependencies:
   ```bash
   cd ui
   npm install
   ```

2. **Start the Development Server:**
   Run the Angular development server:
   ```bash
   npm start
   ```

3. **Access the Application:**
   Open your browser and navigate to `http://localhost:4200/`.
